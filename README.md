# Cardiovascular Disease Risk Prediction System
A machine learning-based web application to assess CVD risk using Random Forest classification.
## Features
* Interactive React Dashboard
* Flask API Backend
* Pre-trained ML Model (Random Forest)
* Comprehensive EDA Notebooks
## Setup
1. Install dependencies: 'pip install -r requirements.txt'
2. Start Backend: 'python backend/app.py'
3. Start Frontend: 'npm run dev'
